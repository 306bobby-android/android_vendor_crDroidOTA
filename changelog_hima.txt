     03-14-2022
====================
Device Maintainer: platinumthis
Device specific changes:

- More undervolting
- Use now Big cores for system and littel cores for backround process, now faster without need to get high clocks on littel cluster. Maximal clock speeds are 0.768 Ghz on littel and 1,2 Ghz on Big one
- Tuned fiops scheduler and use by default
- Set gpu clock from 630 Mhz to 390 Mhz for better battery, without getting lost overall performance. If want true performance you can select cpufreq as GPU governor but i dont recommend it couse it will drain battery quickly and get hot, its not well optimized for now. (in progress)
